# 📂 Gestor de Proyectos
Aplicación **fullstack** para la gestión de proyectos, clientes y tareas.  
Desarrollada como primer proyecto completo de mi carrera en desarrollo web.

## 🚀 Tecnologías utilizadas
- **Backend**: NestJS, PostgreSQL, JWT, Swagger  
- **Frontend**: Angular  
- **Herramientas**: pgAdmin, TypeScript, ESLint, Prettier  

## ⚙️ Instalación y ejecución
### 1. Clonar el repositorio
bash
git clone https://github.com/TU_USUARIO/gestor-de-proyectos.git
cd gestor-de-proyectos

2. Backend
bash
cd backend
npm install
npm start

4. Frontend
bash
cd frontend
npm install
ng serve

El frontend estará disponible en http://localhost:4200  
El backend en http://localhost:3000

📖 Documentación de la API
La API está documentada con Swagger.
Accede en: http://localhost:3000/api

📌 Funcionalidades principales
Registro y login de usuarios con JWT.

Creación y gestión de proyectos.

Asignación de clientes y tareas.

Documentación de endpoints con Swagger.
